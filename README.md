### Getting Started

This project uses [CocoaPods](http://cocoapods.org/).

``` bash
$ git clone https://github.com/popcorntimeios/popcorntimeios.git
$ cd popcorntimeios/
$ pod install
$ open PopcornTime.xcworkspace/
```
